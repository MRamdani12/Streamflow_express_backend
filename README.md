# Task Manager App 🚧 Work in Progress

A full-stack task management application built with Node.js, Express, and PostgreSQL. Deployed on an Ubuntu Server via VirtualBox VPS.

> **Status**: Active development — features and structure may change.

---

## Tech Stack

- **Backend** — Node.js, Express
- **Database** — PostgreSQL
- **Server** — Ubuntu Server (VirtualBox VPS)
- **Auth** — JWT (JSON Web Tokens)

---

## Features

- [x] User authentication (register / login)
- [x] JWT-based session management with refresh token rotation
- [x] Create, update, delete projects

---

## Getting Started

### Prerequisites

- Node.js v18+
- PostgreSQL
- npm

See `.env.example` for all required variables.
